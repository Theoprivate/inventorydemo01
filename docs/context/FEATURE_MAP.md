## Shared flow files

- API route registration and validation: `apps/api/src/app.ts`
- Service rules/orchestration: `apps/api/src/services/inventory-service.ts`, with stock rules in `apps/api/src/services/rules.ts`
- Repository contract: `apps/api/src/repositories/inventory-repository.ts`
- Google Sheets adapter: `apps/api/src/repositories/google-sheets-inventory-repository.ts`
- Sheet models/headers and row mapping: `apps/api/src/models.ts`, `apps/api/src/utils/mappers.ts`
- Browser API wrapper/contracts: `apps/web/lib/api.ts`, `apps/web/lib/types.ts`

Activity/XP/KPI is intentionally absent from the main table: repository/service/schema pieces exist under `apps/api/src/modules/user-activity/` and shared models, but no current HTTP route is registered, so the feature is partial rather than an implemented user flow.
